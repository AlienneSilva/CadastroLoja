Site voltado para cadastro de vendas efetuadas pelos vendedores da loja, afim de controle de estoque, lucro e vendas.
Pagina de cadastro:
* O Usuário será capaz de inserir a informação nome do cliente;
* O usuário será capaz digititar o nome pelo responsavel da venda;
* O usuário informará a quantidade peças/itens vendidos;
* O usuário informará o valor unitário das peças/itens;
* O usuário informará o canal da qual foi extraído a venda;
* O usuário informará por conta cadastrada no canal, é oriundo a venda;
* O usuário será capaz de gravas as informações inseridas;

Ao gravar as informações inseridas, os dados serão enviados posteriomente a uma planilha de edição online, que servirá como Banco de dados, para futuros relátórios gerenciais.

O site foi hospeado no Render, para que fosse possivel o deploy.
